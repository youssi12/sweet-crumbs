# 🍰 Sweet Crumbs Bakery

A modern bakery e-commerce frontend built with React, TypeScript, and Tailwind CSS.

## Quick Start

```bash
unzip sweet-crumbs.zip
cd sweet-crumbs
npm install
npm run dev
```

## Pages

| Route | Description |
|-------|-------------|
| `/` | Homepage |
| `/products` | Product catalog with search & filters |
| `/products/:id` | Product detail page |
| `/cart` | Shopping cart |
| `/checkout` | 4-step checkout flow |
| `/about` | Bakery story & team |
| `/contact` | Contact form & hours |
| `/dashboard` | Customer account |
| `/admin` | Admin dashboard |

## Tech Stack

- **React 18** + **TypeScript**
- **Tailwind CSS** — pink, peach & cream palette
- **Framer Motion** — page & component animations
- **Recharts** — admin revenue & analytics charts
- **React Router v6** — client-side routing
